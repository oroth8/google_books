# Google Books Search

![GoogleBook-ScreenShot](./config/screenshot.png)

A MERN stack React.js application that uses the google books api to find and save books.

## Installation

Use the package manager npm. Install on both root and client side.

```bash
npm i
```

## Usage

```bash
npm run start
```
